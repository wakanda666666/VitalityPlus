
# Vitality+ App Deployment Guide

## How to Host on GitHub Pages
1. Unzip this file on your device.
2. Go to GitHub, create a repository named 'VitalityPlus'.
3. Upload all files (do NOT upload the ZIP itself).
4. Go to Settings > Pages > Enable Pages using branch 'main' and root folder '/'.
5. Save and wait for 1-2 mins. Your live app will be at:
   https://yourusername.github.io/VitalityPlus/

## Install on Mobile
- iPhone: Open the link in Safari > Share > Add to Home Screen.
- Android: Open in Chrome > Menu > Add to Home Screen.
